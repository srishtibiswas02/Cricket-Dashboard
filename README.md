## Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
cd cricket-dashboard
```

## Dependencies

- Python 3.x
- tkinter
- requests
- matplotlib
- numpy
- Pillow (PIL)

## Usage

1. Run the dashboard:
```bash
python dashboard.py
```

2. Select a match from the available live or recent matches
3. Use the sidebar to navigate between different views:
   - Overview
   - Batting Analysis
   - Bowling Analysis
   - Players
   - Match Progress
